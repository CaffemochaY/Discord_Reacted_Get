# Discord_Reacted_Get

ディスコードのリアクションした人を取得する

---

個人的に必要だったため作ったChrome拡張機能  
**ページを読み込んで60秒以内にリアクションのリストを開いた場合**、自動でリアクションした人を取得してクリップボードにコピーする（コンソールにも出力される）

18人までしか取得できない

## 動作ブラウザ

Chrome, Vivaldi, Edge では動作確認しました
